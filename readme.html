Crypto Cafe Payment App

ReadMe

Fully functional cryptocurrency café ordering web app built on the Ethereum Sepolia testnet. It delivers a seamless, web3-inspired ordering experience and features real-time ETH/USD price conversion powered by the CoinGecko API, ensuring transparent, up-to-date pricing for every order.

Add the SepoliaTestnet to your wallet and head to Alchemy Faucets for testnet funds.

Tech Stack

  • Frontend: React 19 (with React Compiler), Vite, TypeScript, React Query, modern responsive UI with touch support
  • Web3 Integration: WalletConnect, Wagmi/Viem, Alchemy
  • Backend & Data: Vercel Serverless Functions (Node.js/TypeScript for backend logic and secure API key management), Supabase
  • API Services: CoinGecko (real-time ETH/USD price conversion)
  • DevOps & Hosting: GitHub (source control), Vercel (CI/CD, scalable hosting for frontend and backend)

Security & Reliability

  • Centralized Security: Sensitive backend credentials and API keys are never exposed to the frontend and are securely managed server-side.
  • User Privacy: The app minimizes data collection and uses encryption for any necessary user data.
  • Wallet Integration: Connect your wallet to start making payments. Supported wallets include MetaMask and other Web3 providers.
  • Best Practices: Regular security audits, dependency updates, and robust error handling are implemented to protect users and transactions.
  • Testnet Deployment: Built on Sepolia for safe experimentation and development.
  • Support: Email: info@blockchaininnovationlab.io
